== Description ==
A simple plugin for show the latest photos of your instagram, you can display on grids or normal size.

Un plugin simple para mostrar las fotos más recientes de tu instagram, puedes mostrarlas en cuadriculas o en un tamaño normal.

* You can set color for grids.
* Choose for grids or normal size
* Color for text
* Set with a shortcode
* Simple options for custom.

== Installation ==
Please, upload the plugin and activate, after, go to options page of the plugin \"Simple Instagram\" and begin customize options.
